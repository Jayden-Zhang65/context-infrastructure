# Skill: Telegram 碎片知识捕获

## When to Use
用户通过 Telegram 发送想法、问题、学习笔记或日常记录时触发。

## 目标
把碎片化的手机输入沉淀为结构化知识，让内容在 PC 端 Claude Code 会话里也能被引用。

## 执行步骤
1. 正常回答用户问题
2. 判断是否值得沉淀（有新观点、有结论、有决策或待办时沉淀）
3. 将值得保留的内容追加到 contexts/daily_records/YYYY-MM-DD.md

## 记录格式
## HH:MM [来源：Telegram]
[简洁标题]
[内容摘要 2-3 句话]
**标签**：#想法 / #学习 / #决策 / #待办 / #Debug策略

## 注意事项
- 只记录有复用价值的内容，不记流水账
